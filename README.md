C’est bon le plugin est carré Voilà ce qu’il faut retenir :

Install rapide : Dès que tu mets le .jar sur un serv, tu tapes juste scsetup <TonPseudo> dans la console. Ça t’ajoute toutes les perms et ça configure les groupes LuckPerms (guide, modo, admin, etc.) en une seconde.

/sc <msg> : Le chat classique.

/scmute : Pour couper le sifflet au chat staff (pratique pour les réunions ou les spams).

/permsc <joueur> : Pour add un staff rapido.

/scsetup <joueur> : (CONSOLE UNIQUEMENT) La commande pour tout configurer d'un coup au début.

Compatibilité : J'ai calé le build en Java 17 pour que ça passe crème sur Velocity 3.4.0, même si t'as le JDK 21 sur ton PC.
